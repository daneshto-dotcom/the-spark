# Off-Model Voltkin Sprites — v1 Archive (S22 P4)

These 5 sprites were generated by the S21 side session as part of the original
Voltkin pose set. After user review on 2026-05-13, they were declared
**off-model** for v1 — the side-session generator produced at least four
stylistically distinct designs across the 6 character poses, breaking the
"strict identity consistency" gate required for an animatable character.

Only `voltkin-zap.png` (Round Looney-Tunes screaming gremlin) was kept as the
**v1 canonical** because it matches the user-attached reference image. The five
poses below ship in v1 ONLY as design history; they are not served from
`public/godly/voltkin/sprites/`.

| File | Design | Why off-model |
|---|---|---|
| `voltkin-idle-1.png` | Boxy-square-head chibi | Different head geometry, body proportions, eye style vs Round-Zap canonical |
| `voltkin-idle-2.png` | Boxy-square-head chibi (running) | Same as idle-1 — boxy lineage |
| `voltkin-charge.png` | Muscular anime-shonen warrior with blue plasma orbs | Completely different art style (shonen-action vs Looney-Tunes-cartoon) |
| `voltkin-victory.png` | Soft pastel cat with stars | Different species. Likely model drift on "victory" prompt |
| `voltkin-hurt.png` | Boxy-square-head chibi (dazed) | Boxy lineage — not Round-Zap |
