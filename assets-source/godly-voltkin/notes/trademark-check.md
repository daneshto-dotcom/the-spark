## Soundalike caveat — "Volkner"

Search surfaced **Volkner**, an electric-type Gym Leader from Pokémon Diamond/Pearl/Platinum (a human NPC, not a creature). The name "Voltkin" is *phonetically adjacent* — same `Vo-` opening, same electric-theme association.

**Risk assessment:** LOW.
- Volkner is a human character name; Voltkin is a creature mascot — different product categories
- Naming convention difference: `-kner` vs `-kin` is clearly distinct in print
- Many electric-themed indie games use `Volt-` prefix without challenge
- The character design lock (Static Gremlin — square gremlin, NOT a mouse) is the actual IP firewall; the name is secondary

**Mitigation:** If commercializing beyond friends/family scope, retest with a trademark attorney. For now, ship it.
