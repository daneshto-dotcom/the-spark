**P1 — Whip wave drift (Micro, commit e85342d)**
Added `const driftPhase = p.tick * 0.022;` inside `drawWhip` and used it in the sin term:
`Math.sin((t * cycles + driftPhase) * Math.PI * 2) * amp`. Wave propagates A→B at one
wavelength every ~2.4s at 60Hz. Closes whip half of S7 PRIME-AUDIT delta. +1 regression
test (`fx.whip differs at tick=0 vs tick=120`).

**P2 — Lattice cross-hatch contrast (Micro, commit 9b0fed8)**
Replaced `width: 1, alpha: p.alpha * 0.5` constants in `drawLattice` cross-hatch with:
`crossWidth = Math.max(1.2, p.width * 0.55)` and `crossAlpha = p.alpha * 0.65`. At HIGH
tier (p.width=3) cross-hatch is now 1.65px vs outline 2.4px (~70% weight; was 42%). Floor
1.2 keeps it crisp at small bond lengths. Closes lattice half of S7 PRIME-AUDIT delta. +1
regression test (`fx.lattice emits 3 strokes`).

**P3 — Warped 3-fold rotation + breathing (Micro, sister fix, commit 9550000)**
drawWarped was named "warped" but sat motionless — sister defect to drawWhip's static sine,
caught by close re-read against the animated-silhouette pattern. Added `rotPhase = p.tick *
0.008` (full turn ~13s) inside `sin(a * 3 + rotPhase)` lobe term, and `breatheAmp = 0.3 +
sin(p.tick * 0.025) * 0.08` (0.22–0.38 lobe extent, ~4.2s cycle) replacing the static 0.3
multiplier. At tick=0 breatheAmp=0.3 — backward-compat with prior visual baseline. +1
regression test (`fx.warped differs at tick=0 vs tick=120`).

**P4 — Filament starburst shimmer + mock extension (Micro, commit 816f965)**
Ray alpha modulates `0.40–0.70` of p.alpha over ~2.6s via `sin(p.tick * 0.04)`. Main bond
stroke unchanged. GraphicsMock.stroke() extended to capture `[width, color, alpha]` args
so alpha-only animations show up in serialize-comparison tests. Mock extension verified
safe across all 35 existing tests (tick-independent silhouettes emit identical stroke args
at any tick → equality tests still pass). +1 regression test (`fx.filament differs at
tick=0 vs tick=40`, near shimmer peak).

**P5 — Static-equality test consolidation (Micro, commit 0b9eaf4)**
